### Тестовое задание от Selectel

Мини-приложение на **Angular 19+** из двух страниц.

#### Что реализовано

1. **Главная страница** — приветствие и ссылка на страницу меню.
2. **Страница меню** — боковое меню с категориями и список элементов с чекбоксами.
3. **Шапка** — отображает текущую категорию, количество выбранных элементов и их общее значение.

#### Стек

- Angular 19+
- TypeScript
- HTML/CSS
